# 🏮 2026年 新年快乐 - 步步高升 (StepHigher)

一款充满节日氛围的跳跃游戏，祝你在新的一年里步步高升！

## 🎮 在线试玩

👉 **[点击这里开始游戏](https://FujimiyaKaor1.github.io/NewYearGame_StepHigher/)**

## 🕹️ 游戏玩法

- **控制方式**：
  - 电脑端：使用鼠标控制角色左右移动。
  - 手机端：滑动屏幕左右两侧。
- **目标**：
  - 跳得越高越好！
  - 收集金币和道具获得加分。
  - 小心不要掉下去哦。

## 🛠️ 本地运行

如果你想在本地运行此游戏：

1. 克隆仓库：
   ```bash
   git clone https://github.com/FujimiyaKaor1/NewYearGame_StepHigher.git
   ```
2. 安装依赖（需要 Node.js）：
   ```bash
   npm install
   ```
3. 启动开发服务器：
   ```bash
   npm start
   ```

## 📱 安装为应用

本项目支持 PWA（渐进式 Web 应用），您可以在浏览器中将其安装到桌面或手机主屏幕，获得原生应用般的体验。

此计划将针对 CSS 进行全面重构，打造沉浸式的国潮新年视觉体验。

## 1. 视觉体系重构 (Visual System)
*   **配色方案 (Palette)**:
    *   主色: 中国红 (`#D6000F`)，象征喜庆。
    *   辅色: 琉璃黄 (`#FFD700`)，象征尊贵与财富。
    *   点缀: 霓虹青 (`#00FFFF`)，保留赛博朋克感。
    *   背景: 深紫渐变 (`linear-gradient(to bottom, #1a0033, #330000)`)，模拟除夕夜空。
*   **纹理与图案**:
    *   使用 CSS `radial-gradient` 和 `mask` 模拟中式窗格纹样。
    *   添加 SVG 内联背景图案（云纹、剪纸）。

## 2. 动态氛围 (Atmosphere & Animation)
*   **飘落特效**: 纯 CSS 实现的雪花 (`.snow`) 和金粉 (`.gold-dust`) 飘落动画。
*   **呼吸与光效**: 按钮和标题增加 `box-shadow` 呼吸灯效，模拟灯笼的光晕。
*   **装饰元素**:
    *   屏幕角落添加垂挂的 CSS 绘制灯笼。
    *   边框增加“春联”风格的装饰条。

## 3. 字体与排版 (Typography)
*   **字体栈**: 优先使用系统衬线体 (`"Songti SC", "SimSun", serif`) 搭配现代无衬线体，营造复古时尚感。
*   **特效**: 增强 `.neon-text` 和 `.glitch-text` 的表现力，使其更具节日张力。

## 4. 响应式与兼容性 (Responsive & Compat)
*   **布局**: 确保所有新增装饰元素使用绝对定位 (`absolute/fixed`) 且 `pointer-events: none`，不遮挡游戏操作区。
*   **媒体查询**: 针对移动端隐藏过于复杂的装饰（如过多的粒子），保证性能。

## 5. 实施步骤
1.  **重写 `style.css`**: 定义新的 CSS 变量和关键帧动画。
2.  **HTML 结构微调**: 在 `index.html` 中插入用于背景特效的容器（如 `<div id="bg-effects">`）。
3.  **文档编写**: 创建 `STYLE_GUIDE.md` 说明新样式的使用。

## 6. 交付物
-   `css/style.css` (完全升级版)。
-   `STYLE_GUIDE.md`。

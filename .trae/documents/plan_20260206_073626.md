此计划将全面升级游戏的节日氛围与玩法体验，融合传统文化与现代游戏性。

## 1. 核心玩法调整：智能平台生成
**问题**: 目前平台随机生成可能导致水平跨度过大，超出角色的跳跃能力。
**解决方案**:
-   **智能生成算法**: 修改 `Game.js` 中的 `generatePlatform`，基于上一个平台的位置计算新平台的位置。
-   **约束条件**: 限制新平台的 X 坐标在 `[prevX - maxJumpDistance, prevX + maxJumpDistance]` 范围内（需考虑屏幕边界回绕）。
-   **Y轴约束**: 确保垂直间距在角色跳跃顶点附近，保证流畅衔接。

## 2. 美术资源升级：福马与新年元素
**角色重绘**: 修改 `Assets.js`，将抽象的圆形角色升级为具体的 Q 版小马。
-   **造型**: 增加马耳、鬃毛细节，身体为喜庆的红色。
-   **装饰**: 身体中央绘制金色的书法“福”字，自带发光特效。
**场景元素**:
-   **背景**: 在 `Game.js` 的 `draw` 方法中，增加静态或缓慢漂浮的灯笼背景层。
-   **特效**: 替换默认粒子为微型鞭炮屑或金粉效果。

## 3. 音频系统：国潮民乐
**合成音乐**: 升级 `AudioController`，使用 Web Audio API 合成更具中国风的旋律。
-   **乐器模拟**: 使用 `triangle` 波模拟笛子，`sawtooth` 波模拟唢呐（低音量）。
-   **旋律库**: 编写一段基于五声音阶（宫商角徵羽）的欢快旋律循环播放。

## 4. UI/UX 升级：游戏说明与风格统一
**开始界面**:
-   **新增模态框**: 点击“游戏说明”按钮弹出，使用红底金边的卷轴样式。
-   **内容**: 包含操作指南（图示）、“小马送福”寓意介绍。
-   **视觉统一**: 所有按钮和弹窗采用中式窗格纹样边框，字体统一为衬线体（宋体/楷体风格）。

## 5. 实施步骤
1.  **算法优化**: 修改 `js/game.js` 平台生成逻辑。
2.  **美术重构**: 修改 `js/assets.js`，重写 `_drawPlayerRaw` 和背景绘制逻辑。
3.  **UI 开发**: 修改 `index.html` 和 `css/style.css`，添加说明弹窗。
4.  **音频增强**: 扩展 `js/audio.js` 的 BGM 编曲。
5.  **文档更新**: 生成资源集成文档。

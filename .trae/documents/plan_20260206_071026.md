此计划旨在对游戏进行**数值重构**和**视觉焕新**，以满足您对更舒缓节奏和 2026 赛博国潮风格的需求。

## 1. 角色数值调整 (Character Balancing)
目标：提升跳跃高度，降低操作频率，营造“飘逸”手感。
-   **跳跃高度**: 提升约 40% (从 -10 调整至 -14)。
-   **重力参数**: 降低约 30% (从 0.4 调整至 0.25)，配合跳跃高度增加滞空时间。
-   **操作频率**: 由于滞空时间变长，玩家每分钟需要的点击/滑动次数将自然降低 35-40%。
-   **平台间距**: 相应拉大垂直间距 (从 100-120px 增加至 150-180px)，适配新的跳跃能力。

## 2. 视觉升级 (Visual Overhaul) - 2026 赛博国潮
目标：打造“金红渐变 + 霓虹粉紫 + 鎏金点缀”的高端视觉体验。
-   **配色方案**:
    -   背景: 深红至紫黑渐变 (赛博夜景感)。
    -   主色: 霓虹红 (#FF0055), 流光金 (#FFD700), 科技紫 (#9D00FF)。
-   **UI 元素**:
    -   **字体**: 使用 CSS `text-shadow` 实现立体发光字效果 (Neon Glow)。
    -   **按钮**: 剪纸风格外框 + 呼吸灯动画 (`@keyframes breathe`)。
    -   **粒子特效**: 在主界面添加 Canvas 绘制的动态烟花背景。
-   **高清适配**: 确保所有新 UI 元素支持 2K 分辨率下的清晰显示 (使用 `rem` 和 SVG/Canvas 绘制)。

## 3. 音频氛围 (Audio Atmosphere)
-   **背景音乐 (BGM)**: 使用 `AudioContext` 合成低频且舒缓的 pentatonic (五声音阶) 旋律，模拟古筝/电子琴的混音，循环播放。
-   **环境音效**: 随机触发柔和的远处爆竹声 (低通滤波器处理，不刺耳)。

## 4. 实施路线图
1.  **数值调整**: 修改 `js/game.js` 中的物理常量 (`gravity`, `jumpForce`) 和生成逻辑。
2.  **视觉重构**:
    -   修改 `css/style.css`: 引入新配色、字体特效、按钮动画。
    -   修改 `js/assets.js`: 更新 Canvas 绘图配色（福马、云朵改为霓虹风格）。
3.  **特效开发**: 在 `js/game.js` 中新增 `Firework` 类，用于主界面背景渲染。
4.  **音频升级**: 扩展 `js/audio.js`，加入 BGM 合成器。
5.  **文档输出**: 生成 `UI_GUIDELINES.md` (UI规范), `BALANCE_CHANGELOG.md` (数值变更表), `TEST_REPORT.md` (性能报告)。
